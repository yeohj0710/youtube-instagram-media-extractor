YouTube·Instagram 미디어 추출기

developed by yeohj0710

YouTube 영상/Shorts 또는 Instagram 릴스/게시물 링크를 넣으면 MP3/MP4 중 원하는 형식으로 빠르게 저장하는 프로그램입니다. 여러 링크를 큐에 넣어 순서대로 처리할 수 있습니다.

사용 순서

  1. YouTube 영상/Shorts 또는 Instagram 릴스 링크를 복사합니다.
  2. 프로그램의 링크 입력칸에 붙여넣습니다. 여러 개는 줄마다 하나씩 넣어도 됩니다.
  3. MP3 또는 MP4 저장 형식을 선택합니다.
  4. 저장 폴더를 확인하거나 변경 버튼으로 바꿉니다.
  5. MP3 큐에 추가 또는 MP4 큐에 추가 버튼을 누릅니다.
  6. 처리 중에도 링크를 더 붙여넣어 큐에 추가할 수 있습니다. 앞 작업부터 차례대로 완료됩니다.
  7. 완료 후 최근 저장 폴더 열기로 저장된 파일을 확인합니다.

저장 위치

기본 저장 위치는 프로그램 폴더 안의 다운로드한 미디어 폴더입니다. MP4는 영상 제목 폴더 안에 MP4 파일과 1초 간격 스크린샷 JPG가 함께 저장됩니다.

참고

OpenAI API 키는 필요 없습니다. Instagram 로그인이 필요한 릴스는 브라우저 로그인 정보가 필요할 수 있습니다.
