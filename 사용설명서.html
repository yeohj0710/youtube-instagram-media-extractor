YouTube·Instagram 미디어 추출기

developed by yeohj0710

YouTube 영상/Shorts, Instagram 릴스/게시물 링크 또는 내 컴퓨터의 영상/오디오 파일을 넣으면 영상과 소리를 원하는 조합으로 빠르게 저장하는 프로그램입니다. 여러 작업을 큐에 넣어 순서대로 처리할 수 있습니다.

사용 순서

  1. 링크로 가져오기 또는 내 컴퓨터 파일 중 하나를 선택합니다.
  2. 링크를 붙여넣거나 PC에 저장된 영상/오디오 파일을 선택합니다.
  3. 영상과 소리를 체크합니다. 둘 다 켜면 소리 포함 MP4, 소리만 켜면 MP3, 영상만 켜면 무음 MP4로 저장됩니다.
  4. 영상 화질과 소리 품질을 선택합니다. 영상을 켠 경우에만 1초 간격으로 이미지 추출 옵션을 추가로 켤 수 있습니다. 로컬 영상에도 동일하게 적용됩니다.
  5. 저장 폴더를 확인하거나 변경으로 바꿉니다. 열기로 바로 폴더를 열 수도 있습니다.
  6. 큐에 추가하고 추출 시작 버튼을 누릅니다.
  7. 처리 중에도 새 링크나 파일을 하나씩 큐에 추가할 수 있습니다. 앞 작업부터 차례대로 완료됩니다.
  8. 완료 후 저장 폴더 열기로 저장된 파일을 확인합니다.

저장 위치

기본 저장 위치는 프로그램 폴더 안의 다운로드한 미디어 폴더입니다. MP3와 MP4 파일은 이 폴더 바로 안에 저장됩니다. 1초 간격으로 이미지 추출을 켠 작업만 영상 제목 스크린샷 추출본 폴더가 별도로 만들어지고, 그 안에는 캡처 JPG만 저장됩니다.

참고

OpenAI API 키는 필요 없습니다. Instagram 로그인이 필요한 릴스는 브라우저 로그인 정보가 필요할 수 있습니다. 기본으로 켜져 있으며, 프로그램이 Chrome/Edge 등 브라우저 프로필을 자동으로 순서대로 확인합니다.

Chrome 보안 정책이나 회사 PC 설정 때문에 자동 쿠키 읽기가 막히면, 브라우저 확장 프로그램으로 Instagram cookies.txt를 내보낸 뒤 고급 옵션의 쿠키 파일에 선택해 주세요.
