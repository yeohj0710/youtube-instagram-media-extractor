# YouTube·Instagram 미디어 추출기

YouTube 영상/Shorts, Instagram 릴스/게시물 링크 또는 내 컴퓨터의 영상/오디오 파일을 넣으면 영상과 소리를 원하는 조합으로 빠르게 저장하는 Windows 프로그램입니다.

**developed by yeohj0710**

## 다운로드

1. 이 페이지 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
2. `Download ZIP`을 누릅니다.
3. 내려받은 ZIP 파일의 압축을 풉니다.
4. 폴더 안의 `YouTube·Instagram 미디어 추출기.exe`를 실행합니다.

![GitHub Download ZIP 위치](프로그램%20구성%20파일/github-download-zip.png)

## 사용 순서

1. `YouTube·Instagram 미디어 추출기.exe`를 실행합니다.
2. `링크로 가져오기` 또는 `내 컴퓨터 파일` 중 하나를 선택합니다.
3. 링크를 붙여넣거나 PC에 저장된 영상/오디오 파일을 선택합니다.
4. `영상`과 `소리`를 체크합니다. 둘 다 켜면 소리 포함 MP4, 소리만 켜면 MP3, 영상만 켜면 무음 MP4로 저장됩니다.
5. 영상 화질과 소리 품질을 선택합니다.
6. `영상`을 켠 경우에는 `1초 간격으로 이미지 추출` 옵션을 추가로 켤 수 있습니다. 로컬 영상에도 동일하게 적용됩니다.
7. 저장 폴더를 확인하거나 `변경`으로 바꿉니다. `열기`로 바로 폴더를 열 수도 있습니다.
8. `큐에 추가하고 추출 시작`을 누릅니다.
9. 처리 중에도 새 링크나 파일을 하나씩 큐에 추가할 수 있습니다. 앞 작업부터 차례대로 완료됩니다.
10. 완료되면 `저장 폴더 열기`로 결과를 확인합니다.

## 결과 파일

기본 저장 위치는 프로그램 폴더 안의 `다운로드한 미디어` 폴더입니다.

MP3와 MP4 파일은 `다운로드한 미디어` 폴더 바로 안에 저장됩니다.

```text
다운로드한 미디어/
  2605122101 영상제목.mp4
  2605122102 영상제목.mp3
```

`1초 간격으로 이미지 추출`을 켠 작업은 스크린샷 폴더가 별도로 만들어집니다.

```text
다운로드한 미디어/
  2605122101 영상제목.mp4
  영상제목 스크린샷 추출본/
    0001_00-00-00.jpg
    0002_00-00-01.jpg
    0003_00-00-02.jpg
```

## 참고

OpenAI API 키는 필요 없습니다.

Instagram 로그인이 필요한 릴스는 브라우저 로그인 정보가 필요할 수 있습니다. 기본으로 켜져 있으며, 프로그램이 Chrome/Edge 등 브라우저 프로필을 자동으로 순서대로 확인합니다.

Chrome 보안 정책이나 회사 PC 설정 때문에 자동 쿠키 읽기가 막히면, 브라우저 확장 프로그램으로 Instagram `cookies.txt`를 내보낸 뒤 프로그램의 고급 옵션에서 `쿠키 파일`로 선택해 주세요.
