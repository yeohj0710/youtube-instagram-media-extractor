# YouTube·Instagram 미디어 추출기

YouTube 영상/Shorts 또는 Instagram 릴스/게시물 링크를 넣으면 영상과 소리를 원하는 조합으로 빠르게 저장하는 Windows 프로그램입니다.

**developed by yeohj0710**

## 다운로드

1. 이 페이지 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
2. `Download ZIP`을 누릅니다.
3. 내려받은 ZIP 파일의 압축을 풉니다.
4. 폴더 안의 `YouTube·Instagram 미디어 추출기.exe`를 실행합니다.

![GitHub Download ZIP 위치](프로그램%20구성%20파일/github-download-zip.png)

## 사용 순서

1. `YouTube·Instagram 미디어 추출기.exe`를 실행합니다.
2. YouTube 영상/Shorts 또는 Instagram 릴스 링크 하나를 붙여넣습니다.
3. `영상`과 `소리`를 체크합니다. 둘 다 켜면 소리 포함 MP4, 소리만 켜면 MP3, 영상만 켜면 무음 MP4로 저장됩니다.
4. 영상 화질과 소리 품질을 선택합니다.
5. `영상`을 켠 경우에는 `1초 간격으로 이미지 추출` 옵션을 추가로 켤 수 있습니다.
6. 저장 폴더를 확인하거나 `변경`으로 바꿉니다. `열기`로 바로 폴더를 열 수도 있습니다.
7. `큐에 추가하고 추출 시작`을 누릅니다.
8. 처리 중에도 새 링크를 하나씩 큐에 추가할 수 있습니다. 앞 작업부터 차례대로 완료됩니다.
9. 완료되면 `저장 폴더 열기`로 결과를 확인합니다.

## 결과 파일

기본 저장 위치는 프로그램 폴더 안의 `다운로드한 미디어` 폴더입니다.

MP3와 MP4 파일은 `다운로드한 미디어` 폴더 바로 안에 저장됩니다.

```text
다운로드한 미디어/
  2605122101 영상제목.mp4
  2605122102 영상제목.mp3
```

`1초 간격으로 이미지 추출`을 켠 작업은 스크린샷 폴더가 별도로 만들어집니다.

```text
다운로드한 미디어/
  2605122101 영상제목.mp4
  영상제목 스크린샷 추출본/
    0001_00-00-00.jpg
    0002_00-00-01.jpg
    0003_00-00-02.jpg
```

- Instagram 게시물

## 참고

OpenAI API 키는 필요 없습니다.

Instagram 로그인이 필요한 릴스는 브라우저 로그인 정보가 필요할 수 있습니다. 기본으로 켜져 있으며, 프로그램 안의 브라우저 로그인 정보 설정에서 변경할 수 있습니다.
